Release Notes
Specify 6.5.03
27 November 2013
Specify Software Project


This update does not change the Specify database schema.

List of changes in Specify 6.5.03

  1. An error no longer occurs under certain conditions when editing group members, collectors, or authors. 11/27/13 10:16 AM (Bugzilla 9449)
  2. GroupNumber is now accessible for queries. 11/26/13 09:53 AM (9451)
  3. Single field formats with added text now display completely in query results. 11/21/13 10:35 AM (9429)
  4. Agents which are associated with data records in other tables, can no longer be deleted. 11/20/13 12:52 PM (9447)
  5. The extra "between" operator in the Query Builder has been removed for databases with non-numeric catalog numbers. 11/14/13 02:59 PM (9446)
  6. Catalog numbers which exceed 9 digits can now be added. 11/14/13 02:40 PM (9199)
  7. Lat/long data is now displayed in the correct location on the map within the map view of the web portal. (Web portal fix.) 11/14/13 10:39 AM (9435)
