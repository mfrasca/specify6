Release Notes
Specify 6.2.02
17 August 2010
Specify Software Project


Highlights for this release:

  1. The uploader is now able to pick to which table attachments apply.
  2. Resource Import/Export can import and export by user.
  3. The Preparation formview now has the option of showing to which loans its preparations belong. This feature requires a manual change of the formsystem's xml (using Resource Import/Export).
  4. The SVN number of each release now appears in Specify's "About" dialog.
  5. Specify has also been localized in Portuguese.
