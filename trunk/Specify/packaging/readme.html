Release Notes
Specify 6.3.05
01 September 2011
Specify Software Project



List of changes in Specify 6.3.05

  1. Two users can now simultaneously edit the same user defined picklist without the risk of exceptions.  2011-09-01  (8527)
  2. Opening a WorkBench data set with associated images no longer causes an exception.  2011-09-01  (8525)
  3. An issue causing improper validation of required fields in the WorkBench uploader has been fixed.  2011-09-01  (8523)
  4. Preparation Description 2 is no longer mapped to Preparation Description 1 in the WorkBench uploader.   2011-09-01  (8522)
  5. When importing a data set into the WorkBench from Excel, formula cells are now imported as the resultant value.  2011-09-01  (8521)
  6. The Import File Splitter no longer changes the date format of data when splitting the file.  2011-09-01  (8509)
