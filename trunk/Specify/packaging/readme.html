Specify 6.3.01

List of changes in Specify 6.3.01

  1. For non-root IT users, Specify has improved its capability to discover the privileges needed to update the database schema.  2011-06-14  (Bugzilla #8425).
  2. The error that appeared when users clicked "MySQL" in Preferences has been removed.  2011-06-14  (8424).
  3. The name and remarks fields for Collection Relationships now save changes correctly.  2011-06-14  (8422).
