New Agent Feature:

Agents of any type are now supported in the WorkBench for uploading. The Uploader no longer assumes agents are persons when checking for duplication against existing agents in a Specify database.  Last Name fields labeled  ' xxx Last Name' where xxx=Collector, Author, Sender, etc. can be used for Group, Organization and Other names.
