Security Tools

Users, Collections, Disciplines and Divisions are added to the database structure and permissions are set in Security Tools; System > Collection Setup > Security Tools.

Security Heirarchy

  • Institution -- (URI is another word for URL)
  • Division
  • Discipline
  • Collection

Users are added at both the Discipline and Collection level. At the Collection level permissions are set for individual collections; whereas users can be given permissions for all Collections found in the Discipline when set at the Discipline level.

 

All users

  • Must be an Agent
  • Belong to a Group (therefore, the group must first be added)
