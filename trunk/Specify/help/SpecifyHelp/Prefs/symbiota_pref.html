Symbiota

New in Specify 6.5.05, data can be shared between Symbiota and Specify through an automated tool within Specify, but first the base URL for the Symbiota project must be established within Specify. This process is completed in following Preferences dialog:

  • For Mac: Choose Specify > Preferences >Symbiota.
  • For Windows and Linux: Choose Edit > Preferences >Symbiota.

Type or Copy/Paste the base URL for your Symbiota project (this must be provided by Symbiota) into the Base URL text box.
