Taskbar Preferences

  • For Mac: Choose Specify > Preferences > Taskbar.
  • For Windows and Linux: Choose Edit > Preferences > Taskbar.

Add and/or remove tools from the Specify Taskbar.

Taskbar Preference
Result
Add Interactions
 Determines whether the Interactions item is visible on the Taskbar and therefore usable in the application. Uncheck Interactions if not using Accession, Permit, Loan, Gift, Exchange In, Exchange Out, Borrow or Information Request tools.
Add Plugins

Places a Plugins button on the Taskbar, which allows individual records or Record Sets to be utilized when using the Plugin services. Google Earth and one of the georeferencing services are available on the Locality form and will remain usable if the Plugins button is removed from the Taskbar.

Uncheck Plugins if not using GoogleEarth, BioGeomancer or GEOLocate other than in the Locality form.
Add Welcome
 Adds a Welcome button to the Taskbar which opens a statistics page when clicked.
