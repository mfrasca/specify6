Email

  • For Mac: Choose Specify > Preferences > Email.
  • For Windows and Linux: Choose Edit > Preferences > Email.

Setting up an email account with Specify is useful for sending information that has been requested by colleagues. The information can be returned as an Information Request and emailed with the click of a button.

To set up the local Email account first click the Account Type button and choose In Use. When the Account Type is Not In Use the information in the dialog is retained, but the email function is turned off.

Next, fill in the following information:

  • SMTP
  • Port
  • Security
  • Username
  • Password
  • Email; (this refers to the email address)

This information can be found in the email provider's Outgoing Mail Sever settings.

Note: The Specify email feature is not used for receiving or checking email. The information provided must be for the Outgoing Mail Server.
