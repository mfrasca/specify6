Login Dialog

  • For Mac: Choose Specify > Preferences > Login Dialog.
  • For Windows and Linux: Choose Edit > Preferences > Login Dialog.

Specify automatically creates a pick list of all Usernames and Database Names that are in use for the Login Dialog. The following buttons delete the names in these pick lists.

Login Dialog Button
Result
Action
Username List
 * The entire list of Usernames will be deleted. Click the Clear button.
Database Name List
 * The entire list of Database Names is deleted. Click the Clear button.
Server Name List
 The Server Name is deleted. Click the Clear button.

* These names are used locally.
