Trees

  • For Mac: Choose Specify > Preferences > Trees.
  • For Windows and Linux: Choose Edit > Preferences > Trees.

This page is used to customize the appropriate Trees in Specify 6.

Tree Preference
Result
Action
Restore Epansion State
 Trees will open to the last node that was selected. If the tree is expanded, but no node is selected the tree will open to the default setting. Check the box.
Show Counts Below
 Determines the level at which counts are shown in the trees. Counts display next to the corresponding node as two numbers (X,Y). X refers to the number of Collection Objects for the node only. Y refers to the number of Collection Objects for the node (parent) and all nodes beneath it (children). Choose a Rank from the drop down menu.
Column Color #1
 Changes the background color of odd columns of the tree. Click the color square and choose a color from the resulting tile box.
Column Color #2
 Changes the background color of even columns of the tree. Click the color square and choose a color from the resulting tile box.
Synonymy Color
 Changes the color of synonymized Tree nodes. Click the color square and choose a color from the resulting tile box.
