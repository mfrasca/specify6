Institution Hierarchy

The Institution Hierarchy should be configured to include the Divisions, Disciplines and Collections that will share one database within an Institution after running the Setup Wizard. It can also be edited at a later date to add or delete any of these institution levels.

Other Specify modules can be open while adding to the hierarchy, so begin by closing all tabs.Tools for adding Divisions, Disciplines and Collections are found by clicking System > Collection Setup > Configuration.

Start adding to the Institution Hierarchy at the Division level and work your way down to Collection. Each level is added at the level above it. For instance, Divisions are added to the Institution form. Click the Institution button on the Side Bar to add Divisions.

Add Divisions

At least one Division should appear in the Division sub form of the Institution form.

Click the (add) button to add a Division.

Name is the only required field and is used in any lists within Specify as well as the Institution Hierarchy.

Alt Name provides another field for name. This is often useful when a particular Division may have changed names.

URI stands for Uniform Resource Identifier and refers to a resource on the internet (a Division web site).

Is Accession Bound tells Specify that this Division does not share Accessions within the institution will have their own accession table and numbers.

 

Add Disciplines

Click the Division button on the Side Bar to add Disciplines.

The Division form will filled-in information will appear and include a Discipline sub form. Use the navigation control at the bottom of the window to step to the correct Division for the Discipline that you wish to add.

In the Discipline sub form click the (add) button to add a Discipline.

A dialog listing all available Discipline types will display. Choose a discipline from the list. The chosen type will appear in the Type field.

Data Type indicates that the data base is for biological collections and is not editable.

Name allows the Discipline to be named anything. This name will display in any Discipline lists within Specify and on the Institution Hierarchy.

 

Add Collections

Collections are added in the Discipline form. Click the Discipline button on the Side Bar.

The Discipline information will appear at the top of the form with a Collections sub form underneath. Use the navigation control at the bottom of the window to step to the correct Division for the Discipline that you wish to add.

Click the (add) button to add a Collection.
