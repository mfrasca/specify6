Google Earth

The WorkBench can generate a Google Earth KML file from row data in a Data St and load it into Google Earth. The rows display as 'push pins' which are identified by their row number. Clicking on the push pin displays the row information as a pop-up window.

Requirements

  • Latitude 1 column heading
  • Longitude 1 column heading
  • Google Earth (http://earth.google.com) installed on the hard drive (after installing a reboot is required)

 

To display records in Google Earth, sweep records in the Data Set to select them and click the Google Earth button (). To map an entire Data Set either select all the records or no records.

Google Earth Results

 

Record locations are marked with a Specify icon.  Click the icon to display the record data from the Data Set.
