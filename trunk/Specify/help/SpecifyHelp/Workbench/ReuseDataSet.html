Reuse Data Set

The WorkBench can create a new Data Set from the structure of any existing Data Set. This eliminates the need to choose the Columns for the new Data Set.

Drag and Drop

The fastest way to create the new Data Set from an existing structure is to 'drag and drop' the Data Set name onto the 'Reuse Data Set' action on the Sidebar. This eliminates the need to choose an existing Data Set to reuse.
