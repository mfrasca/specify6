Reuse DataSet

The WorkBench can create a new DataSet from the structure of any existing DataSet. This eliminates the need to choose the Columns for the new DataSet.

Drag and Drop

The fastest way to create the new DataSet from an existing structure is to 'drag and drop' the DataSet name onto the 'Reuse DataSet' action on the Sidebar. This eliminates the need to choose an existing DataSet to reuse.
