Import Data Overview

WorkBench can easily import data from a MS-Excel spreadsheet or CSV file and use the information to create a DataSet.

The WorkBench not only imports the data, but also maps the existing column headings to fields in Specify 6. This is not only necessary for other features in the WorkBench to function properly, but also ensures that the data will upload correctly into Specify 6.The WorkBench includes an 'automapper' that reads the import column headings and automatically maps them to any matching fields in Specify 6. Any remaining column headings can be manually mapped. The column headings can be mapped to different Specify fields by editing the Data Set Mapping.

The Data Set will display the imported field names rather than the Schema names, as these are more familiar.

If data exists on several scanned card images or field notes, it is very helpful to create a Data Set by importing the images.
