Data Set Backup

Data Sets created in the WorkBench are saved within the DerbyDatabase directory. To back up all the Data Sets simply backup the DerbyDatabase directory to another location. The WorkBench also creates MS Excel backups of each Data Set in the SpecifyWB/Specify/Backups directory each time a Data Set is saved. The Specify directory is the primary data directory for the WorkBench and is the location for preferences and backups.
