Exporting Data

The Workbench can export an entire Data Set or select records within a Data Set by selecting Action items from both the Sidebar and the Workspace Item Bar. The following table describes each of these export functions:

Button Item Item Location Explanation
Export Data Set Sidebar

Choose a closed Data Set

Export the entire Data Set

Export Excel Template Sidebar

Choose a closed Data Set

Export the entire Data Set

Export Data Set Workspace

Active Data Set only

Export select records or the entire Data Set
