Data Set Properties

Data Set Properties includes the name and description of a Data Set.

To view and/or edit the properties of a Data Set, use the right mouse button to click on a Data Set. A context menu appearswithtwo options, select 'Edit Properties' :

The Data Set Properties dialog will appear:

Data Set Properties

 

If a Data Set was created from an imported file, the path to the file will be shown in the 'Source File' area of the dialog.
