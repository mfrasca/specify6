Record Sets

Simple Search and Query both have the ability to return groups of records. These can be named and saved as Record Sets. Saved Record Sets are listed on the Sidebar and can be dragged and dropped onto other items on the Sidebar.

Record Sets will appear in the Sidebar of the following tools:

  • Data Entry
  • Interactions
  • Reports

 

Drag the Record Set upward toward the list of actions (Form Type or Report). A green outline will appear around the action that can accept the data, indicating where to drop the Data Set.

Drop a Record Set onto another Record Set to merge them. Any duplicates will be automatically removed.

 

Record Set Context Menu

Right-button-click on a Records Set to perform the following functions:

  • Rename
  • Delete
  • View; this opens a results window listing the contents of the Record Set and allowing individual records to be deleted.
