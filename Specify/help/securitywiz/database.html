This form tests the validity of the Specify "IT User" name and password. For standalone workstation installations of Specify (without a dedicated MySQL Server computer) the Specify IT User is typically the root user account in the MySQL installation. The user name is "root" and the password is whatever was specified during the MySQL installation on that workstation. You must validate your "IT User" (MySQL "root" user) name and password in this window to be able to use this tool.

At institutions which share a MySQL server for one or more institutional databases, the Specify "IT User" is typically NOT the MySQL root user account, but rather a named user granted several permissions to create and modify Specify databases only-for more information, see the documentation for installing MySQL on a shared server on the Specify Software Project web site.

This version of the Specify DB Security Wizard does not work with Specify IT User names which are not "root". If you are having a problem connecting a Specify installation on a workstation to a dedicated MySQL server and your Specify IT User in MySQL is not "root", please contact the Specify Software Project for assistance.
