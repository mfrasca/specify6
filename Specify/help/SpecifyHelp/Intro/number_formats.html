Attachments

If a path does not exist or Specify is not able to connect to the path the Attachment button will become disabled on the forms.

 

Attachments to records will be stored in this location. It is recommended that this storage area (attachment repository) be on the shared network drive that all users have been given Read/Write/Delete permission for, rather than on a personal (local) computer. If attachments are stored locally they can not be shared.
