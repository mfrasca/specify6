Specify Setup

Once the Specify Database Setup Wizard has been completed a blank database will be attached to Specify. The blank database includes one user and one division, with one discipline within the division, and one collection within the discipline. The database will need to be expanded to include all the necessary users, collections, divisions and disciplines, with each being configured to include schema and tree information.
