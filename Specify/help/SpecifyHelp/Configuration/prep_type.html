Preparation Type

Choose System > Collection Setup > Configuration to edit the Preparation Type.

When entering a Collection Object within a form the Preparation Type is a Pick List that is derived from the Preparation Type table. Since there are no forms for this Table the Preparation Type has its own editing function.

  • Click on the (Prep Type) icon
  • Click the (add) icon to add a Prep Type

    • or
  • Edit an existing Prep Type by using the (Navigation Control) to find the Prep Type.
  • Click the Save button to save changes
