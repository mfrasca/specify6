Query Combo Box View/Edit

The record you have chosen to edit or view has a many-to-one relationship with the primary table. By clicking the edit or view button you opened the record as a sub form.

 

Edit Record

The sub form has the same controls and functionality as a form.

 

View Record

The record can be viewed but not edited. Users with editing priviledges can edit the record by using Simple Search or the Query Builder to find the record, then clicking the (form) button in the colored results bar to open the record in an editable form.
