Import/Export

At this time Specify 6 does not include a Form Editor, as was found in Specify 5 Customize Forms. The forms have been created programmatically to be extremely flexible and can be exported as xml and either edited by on-site staff or emailed to the Specify staff for assistance with form editing and customization. Forms are imported and exported using the Resource Import/Export dialog. Forms are exported as a set of forms and need to be reimported as a set as well.

Reports can also be imported and exported through the use of this tool. Reports are created from a query and export as a file that contains both the report and query. Reports are typically exported so they may be imported back into Specify by either a new user or at a different level, allowing them to be shared within the community.

Choosing System > Resource Import/Export to open the Import/Export dialog. A tab at the top of the tool allows the choice of importing/exporting either a Form Set or Report.

Importing and exporting is based on the same levels that are found within the institution, as well as two additional levels:

  • User
  • User Type
  • Collection
  • Discipline
  • Division
  • Institution
  • System/Backup

For each level the imported resource (form or report) is made available to other users at the same level as well as all levels above it. This is based on the permissions for the person importing. Resources imported at the top (User) levelr will be made available to the user only. At the User Type level the form/report is accessible to all users within the collection at the same User Type as the person importing. For example, if a user imports a report from a tissue collection within the Lichen discipline of Botany into the Division level the report will be available to any user within any collection in the Lichen discipline.

Resources can not be imported or exported from the System level. It holds the resources that Specify ships with and can be used to restore these default resources if needed.

 

Export

  1. Choose the level at which the resource is located.
  2. Choose the resource to export.
  3. Click the Export button.
  4. Choose a name and path in the resulting file system dialog.

 

Import

  • Choose the level to import the resource into.
  • Click the Import button.
  • Choose the file in the resulting file system dialog.

 

Revert

Revert will delete ALL files that have ever been imported. The Uploader tool remembers the files that Specify shipped with at each level and reverts back to those files. If the Uploader recognizes imported files the Revert button will become enabled.
