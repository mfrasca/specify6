Import/Export

At this time Specify 6 does not include a Form Editor, as was found in Specify 5 Customize Forms. The forms have been created programmatically to be extremely flexible and can be exported as xml and either edited by on-site staff or emailed to the Specify staff for assistance with form editing and customization. Forms are imported and exported using the Resource Import/Export dialog.

Other Resources can also be imported and exported through the use of this tool.

Choosing System > Resource Import/Export to open the Import/Export dialog. This includes reports and some system files such as Pick Lists and Web Links files.

Specify implements a heirachical system which limits the accesibility of the forms/resources. This allows forms/resources to be changed and then implemented for those users who wish to use them. When accessing forms/resources, Specify starts at the Personal level, takes the forms/resources, then moves to the next level. When duplicate forms/resources are encountered those from the first level are kept and the higher level forms/resources are ignored.

Level Access Import Export
Personal Accessed by the user only. Yes Yes
User Type Accessed by the user and User Type. Yes Yes
Collection Accessed by the user, User Type and anyone in the collection. Yes Yes
Discipline Accessed by the user, user type, and anyone in the discipline. Yes Yes
Common Accessed by all users. Yes Yes
Backstop This includes system files for the form system and is available to everyone. No No

Forms and resources can be exported, changed and then imported as a new version of the same form/resource, or saved as a new form/resource and added to the system.

Forms do not export or import individually, but as a form set. The xml file contains all the forms for a particular level. Resources export as individual files.

 

Export

  1. Choose the level at which the forms are located.
  2. Choose a form set to export.
  3. Click the Export button.
  4. Choose a name and path in the resulting file system dialog.

 

Import

  • Choose the level to import the forms into.
  • Click the Import button.
  • Choose the file in the resulting file system dialog.

 

Revert

Revert will delete ALL files that have ever been imported. The Uploader tool remembers the files that Specify shipped with at each level and reverts back to those files. If the Uploader recognizes imported files the Revert button will become enabled.
