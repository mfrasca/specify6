Import Data Overview

The WorkBench can easily create a Data Set by importing data from an MS Excel or CSV file.

The WorkBench not only imports the data, but also maps the existing column headings to fields in Specify 6. This is not only necessary for other features in the WorkBench to function properly, but also ensures that the data will upload correctly into Specify 6.The WorkBench includes an 'auto-mapping' feature that reads the import column headings and automatically maps them to any matching fields. Any remaining unmapped column headings can then be manually mapped. The column headings can be mapped to different Specify fields by editing the Data Set Mapping.

The Data Set will display the imported field names rather than the Specify field names, as these are more familiar.

If data exists on several scanned card images or field notes, it is very helpful to create a Data Set by importing the images.
