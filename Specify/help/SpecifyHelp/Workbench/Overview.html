Welcome to WorkBench

The Specify 6 WorkBench is a friendly and innovative software tool for capturing and editing collections data. It can create sets of specimen and lot information through keystroke data entry and through batch data import.

The WorkBench's versatility makes it perfect for use in the field or for rapidly entering a backlog of collections data. Using the WorkBench to enter data ensures that it will not only import seamlessly into Specify 6 but also prepares it for any data processing needs that may occur in the future.

The WorkBench also imports existing MS-Excel or CSV data files and converts the data into versions that are recognized by Specify 6. To achieve the conversion, the WorkBench creates relationships between the existing data Columns and the Specify 6 fields. Once these relationships are defined, or mapped, data can be entered, changed and manipulated using the features of the WorkBench and Specify 6.

The WorkBench is a highly intuitive and streamlined tool for working with data. New users require little training and potential errors are caught before data is uploaded into Specify 6.

Data inside the WorkBench can be presented in reports, edited using a Georeferencing converter, Georeferenced in BioGeomancer and mapped in Google Earth!

Introducing DataSets

Data is entered, edited, organized and stored in a DataSet.

The DataSet can display and function as a Grid, resembling a spreadsheet, or a Form, resembling a database.

When data is imported, the DataSet will display the imported column names rather than the Specify field names, as these are more familiar.

The WorkBench also has the capability of linking one image to each record within a DataSet. The linked images display in an image window that can be toggled on or off. This is particularly useful when entering data from a scanned catalog card.

Using WorkBench

Before using WorkBench, it is important to understand its components and how they interact with each other. As the diagram below shows, the two major components of WorkBench are the Sidebar and Workspace.


Sidebar

The Sidebar contains Data and Action items:

Action items are used to manipate the Data items. There are four ways to initiate an action:

  1. Click an Action item, then choose a Data item
  2. Click a Data Item, then choose an Action item
  3. Drag an Action item and drop it onto a Data item
  4. Drag a Data item and drop it onto an Action item

Drag and Drop

Many Action items can be drag and dropped onto Data items, or vice versa. Action items that can be drag and dropped will produce a green box on a mouse-over. Action items that can not be drag and dropped will produce a blue box.

Selecting an item and dragging it downward on the Sidebar will produce a green box around all the items that it can be dropped onto.


SideBar Items

Sidebar items are further categorized by what they are; i.e. Actions, Reports, Templates and DataSets.

An explanation of each of these individual items is below:

Import Data Click this to import data from a spreadsheet or cvs file.
Import Card Images Click this to enter data from images.
New DataSet Click this to enter new data that will be defined by an existing Template.
Export Data Click this to create an MS-Excel document from a DataSet.
Export Template Click this to create a predefined MS-Excel document from an existing Template.
DataSet Summary Displays a summary of occurrence counts for one field within the DataSet. This is particularly useful when editing data to validate the input.
Chart Charts one field within a DataSet.
Top 10 Displays the top 10 data records of one field within the DataSet in a pie chart.
DataSets

Provides a list of the available DataSets. Click the name to open the DataSet in the Workspace.

Note: When a DataSet is in use it becomes disabled on the Sidebar. This ensures that the DataSet can not change while it is in use.

 

Trash Can

Delete any Template or DataSet by dragging it to the trash can.

Note: Deleting a Template will also delete all the DataSets defined by the Template.


WorkSpace

DataSets and Reports open in the Workspace. The components of the Workspace are listed below:


Tabs

WorkBench allows many DataSets to be open at the same time, but only one can display in the Workspace. WorkBench creates a tab for each open DataSet, allowing users to select which will be displayed.


Close Workspace

Select the 'x' to close an open DataSet or Report.


Getting Started

How to begin depends on the state of the data. If the data exists as an MS-Excel or CSV file then proceed to 'Import Data'. If the data exists on written notes proceed to 'New DataSet '. If the data exists on scanned cards or field notes it may be useful to read the help section 'Import Images' before proceeding to 'New DataSet '.
