Data Set Properties

Data Set Properties includes the name and description of a Data Set.

To view and/or edit the properties of a Data Set, use the right mouse button to click on the Data Set. 'An Edit Properties' context menu appears:

Click the 'Edit Properties' button with the left mouse button to open the Data Set Properties dialog:

Data Set Properties

 

If a Data Set was created from an imported file, the path to the file will show in the 'Source File' area of the dialog.
