The WorkBench is a staging area for pre-cataloged collections data and is not is not intended as replacement for editing for MS-Excel row and column data. As a staging area, the WorkBench provides tools for preparing the data for Specify 6. The most import function is "linking" or "mapping" the data columns from the imported data files to existing data fields within Specify 6. This important step ensures that all the imported will be corectly and systemaically up loaded into Specify 6.

 

Form Editing

Form Editing enables the Data Set rows to be displayed and edited row by row (record by record). Each Column in the grid is displayed with a label and a field.

At the bottom of the form there are several UI controls:

Record Number Control l< Go to the first record
< Go to the previous record
> Go to the next record
>l Go to the last record
Add a record and display it

Show Images Toggles the Image Window on and off.
Configure Carry Forward Opens the 'Carry Forward' configuration dialog. See Carry Forward.

The Label and Field controls can be moved and repositioned on the form with a left mouse button click and drag. The Tab Order is automatically adjusted depending on the placement of the controls, the tab order is top, down, left to right.

Edit Form Properties

In addition to dragging and dropping the controls on the form the form controls have several properties that can manually updated. Click on the label or the field and then the edit properties button. The following dialog will appear:

Edit Form Properties

The text fields in the dialog are explained below:

X Refers to the X coordinate where the text area appears within the Form.
Y Refers to the Y coordinate where the text area appears within the Form.
Label Refers to the column name. This is the only place in the WorkBench that allows a Column name to be changed.
Number of Rows Lists the number of rows the text field will display.
Field Type

Text Field: limits the text area to one row

Text Area: allows the text area to expand to more than one row -- activates the Number of Rows field

Field Columns Refers to the length of the value box.
Help This button opens the Help window.
Apply Displays the change without saving it.
Cancel Cancels the changes that have been made in the dialog.

Carry Forward

Carry Forward enables the value in one or more fields (or columns) within the current record to be 'carried forward' or copied to the next new row of data. Clicking on the button displays a configuration window that enables the columns to be selected.


Configure Carry Forward

The values in the selected fields for the record are carried forward when a new record is created.

Quick Tips for Entering Data

  1. Click on the Data Set to open it
  2. Click in a particular grid cell to enter data and press the esc or F2 buttons to start editing
  3. You are in Grid View, may choose Form View from the icon at the bottom right of the window
  4. You may click and drag field names to rearrange the window
  5. Click or type <cntl> + n to create a new row (or Command + N on the Mac)
  6. Click inside the text field to enter data
  7. Save the Data Set
