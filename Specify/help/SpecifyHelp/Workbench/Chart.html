Chart

The easiest way to create this report is to 'drag and drop' a Data Set onto the Chart action on the sidebar. This eliminates the need to choose a Data Set for the report.

Or, click the Chart action on the Sidebar. A file dialog will appear to choose a Data Set for the report.

Next, choose a column for the report:

Select a Column

 

Select a column and click 'OK'. The report will open in the Workspace.
