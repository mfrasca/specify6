New Template

Templates can be created without importing a MS-Excel or CSV file. Click on "New Template" action in the SideBar and an empty Template Editor dialog will appear.

The window is divided into three sections: Schema, Data Objects and Fields.

Schema refers to the Fields to be included in the New Template.

Data Objects refers to the type of data that is available.

Fields are the names of the fields available for each Data Object.

The Template is filled by choosing a Field within a Data Object and moving them into the Schema. To include a Field, first click on a specific Data Object and the available Fields for that Data Object will display in the Fields box. Next choose a field and and click on the button below the Schema list. The Field will appear in the Schema. A green checkmark will appear in the Fields window showing that the Field has been used. A Field may only appear once in the Schema.

To delete a Field from the Schema:

  1. Select the Field name within the Schema
  2. Select the (-) button

Template Properties

Once the Template has the desired fields click OK to enter a unique name and remarks. The following dialog will be displayed. The blue background in the 'name' field indicates that the 'name' is a required field, and text for the 'remarks' field is optional.

Once the Template information is complete click the OK button to save the Template. For convience an empty DataSet is created with the same name as the Templateand will open in the Workspace.

Quick Tip for Creating a New Template

  1. Click New Template
  2. Click a Data Object
  3. Click the Fields within the data object
  4. Click the ‘+’ to add Fields
  5. Click the (-) button to delete Fields
  6. Save the Template
  7. Name and Describe the Template
