Google Earth

Generates a Google Earth KML file from the row data and loads it into Google Earth. The rows display as 'push pins' which are identified by their row number. Clicking on the push pin displays the row information as a pop-up window.

Requirements

  • latitude1column heading
  • longitude1 column heading
  • Google Earth installed on the hard drive (after installing a reboot is required)

 

To display records in Google Earth, sweep records in a Data Set to select them and click the Google Earth button ( ). To map an entire Data Set either select all the records or no records.

Record locations are marked with a Specify icon.  Clicking on one of the icons presents a 'balloon' containing that record's data.  The balloon title shows the record row number.  The balloon content is a list of column headings and column contents for the record.
