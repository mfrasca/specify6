Taskbar

  • For Mac: Choose Specify > Preferences > Taskbar.
  • For Windows and Linux: Choose Edit > Preferences > Taskbar.

Add and/or remove tools from the Specify Taskbar.

Taskbar Preference
Result
Action
Add Interactions
 Determines whether the Interactions item is visible on the Task Bar and therefor usable in the application. Uncheck Interactions if not using Accession, Permit, Loan, Gift, Exchange In, Exchange Out, Borrow or Information Request tools. Check box.
Add Plugins
Determines whether the Interactions item is visible on the Task Bar and therefor usable in the application. Uncheck Plugins if not using GoogleEarth, BioGeomancer or GEOLocate. Check box.
Add Welcome
 Adds a Welcome icon to the Taskbar which opens a stastics page when clicked. Check box.
